# 1. Проверка email-доменов

Скрипт проверяет список email-адресов: наличие MX-записей и SMTP handshake (без отправки письма).

## Зависимости

```bash
pip install dnspython
```

Python 3.8+.

## Запуск

**Список адресов аргументами:**

```bash
python email_verify.py user1@gmail.com user2@yahoo.com
```

**Список из файла (по одному адресу на строку):**

```bash
python email_verify.py --file emails.txt
```

## Вывод

Для каждого адреса одна строка: `email<TAB>статус`.

Статусы:
- **домен валиден** — MX есть, SMTP принял RCPT TO;
- **домен отсутствует** — MX есть, но сервер отклонил получателя или ошибка при проверке;
- **MX-записи отсутствуют или некорректны** — домен не резолвится, нет MX или неверный формат email.
